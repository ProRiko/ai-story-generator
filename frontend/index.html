How This Works:

  • This web app (frontend) communicates with a **separate Python backend server** running on your local machine.
  • The backend handles secure API calls to generative AI models (Gemini for stories, etc.).
  • **Video generation is conceptual** in the backend; actual video file creation and social media posting requires installing tools like FFmpeg and integrating with cloud services on your backend server.
  • Ensure your Python backend is running before using the "Generate Story" or "Generate Video" buttons.
